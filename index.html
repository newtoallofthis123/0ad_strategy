Infrastructure

Magazine Direction

Editorial code

Preserve

The volume mastheads, thick rules, mono typography, black-paper-accent palette, tactical tables, diagrams, and endplates should stay as the core identity.

Amplify

The website shell can add issue navigation, cover treatments, pull quotes, article decks, and a sharper table-of-contents rhythm around the existing manuals.

Avoid

No soft cards, glossy gradients, generic gaming art, or blog-like reading chrome. The manuals should feel like printed tactical matter placed on the web.
